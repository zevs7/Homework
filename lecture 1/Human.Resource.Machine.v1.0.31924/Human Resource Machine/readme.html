Controls

Everything is controlled by mouse and the mouse button. Press 'f' or ALT+ENTER to toggle fullscreen.



Save and Config Files

Save and config files are located in the following location:

Windows Vista, Windows 7, and later:
C:\Users\[username]\AppData\Roaming\Human Resource Machine\

Windows XP:
C:\Documents and Settings\[username]\Application Data\Human Resource Machine\
